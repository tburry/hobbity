# The Toad Stompers

A campaign site for a hobbit-only TTRPG campaign run by friends. Session recaps, character sheets, NPC records, and a gallery of illustrations.

Built with [Astro](https://astro.build).

## Setup

```sh
pnpm install
pnpm dev
```

## Scripts

- `pnpm dev` — start dev server
- `pnpm build` — build for production
- `pnpm preview` — preview the build
- `scripts/make-avatar <image>` — generate a square avatar from a PC portrait
- `scripts/transcribe-craig <dir>` — transcribe Craig audio tracks

## Contributing

This is a private project for friends at the table. If you're one of them, feel free to open a PR. If you're not, you're welcome to poke around, but the content is specific to our campaign.

PC data lives in `src/content/pcs/`. Stories live in `src/content/stories/`. Images go in `src/assets/images/`.
