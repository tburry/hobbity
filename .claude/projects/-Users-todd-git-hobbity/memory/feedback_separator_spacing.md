---
name: Separator spacing preference
description: User prefers space on either side of separator spans in breadcrumbs and similar UI
type: feedback
---

Always use space on either side of separator spans (e.g. breadcrumb separators). Use padding or margin, not `margin-inline`.

**Why:** User preference for visual spacing consistency.
**How to apply:** When writing CSS for separator elements, use `padding-inline` or explicit left/right margins with enough space to breathe.
